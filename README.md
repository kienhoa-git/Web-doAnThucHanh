# Sakura Shop 🌸

> 🌐 **Trang web:** [kienhoa-git.github.io/Web-doAnThucHanh](https://kienhoa-git.github.io/Web-doAnThucHanh)

**Sakura Shop** là trang web thương mại điện tử được xây dựng bằng HTML, CSS và JavaScript thuần, với giao diện hiện đại, tinh tế lấy cảm hứng từ hoa anh đào.

## 🛍️ Tính năng

- Hiển thị danh sách sản phẩm (Điện thoại, Laptop, Phụ kiện)
- Lọc sản phẩm theo danh mục
- Giỏ hàng với chức năng thêm/sửa/xóa
- Tính tổng tiền tự động
- Lưu giỏ hàng bằng LocalStorage
- Hiệu ứng hoa anh đào rơi (Sakura Petals)
- Menu dropdown và navigation mượt mà
- Responsive — tương thích mọi thiết bị
- Thông báo toast khi thao tác

## 🚀 Công nghệ

- HTML5
- CSS3 (Flexbox, Grid, Animation)
- JavaScript (DOM, LocalStorage)

## 📸 Preview

Giao diện chủ đạo với tông hồng pastel, hiệu ứng mờ kính (glassmorphism) và animation nhẹ nhàng — mang đến trải nghiệm mua sắm dễ chịu cho người dùng.
